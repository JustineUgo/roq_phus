# roq_phus


## Tools/Packages

- State management `Bloc`
- Dependency Injection `Injectable`
- Flutter `3.24`

## Installation

To get started with the project, follow these steps:

2. Clone the code:
   ```bash
   $ git clone https://github.com/JustineUgo/roq_phus.git
   $ cd roq_phus
    ```

2. Run the following commands:
   ```bash
   $ make get
   $ make codegen
    ```
3. Run app:
   ```bash
   $ flutter run
    ```
